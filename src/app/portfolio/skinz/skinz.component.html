Web Development

Skinz is a Manchester-based company that specialises in vehicle enhancement. Roof wraps, vehicle signage, allow/brake calliper refurbishments, window and light tinting, dechroming, and vehicle wraps are among the services they provide.

Client: Skinz
Technologies: Wordpress, ACF, Contact Form 7

I was asked to design a website that would allow users to learn more about the services that Skinz offers. It was also critical that users be able to get answers to their questions about the services they needed quickly.

I created a clean, vibrant website using a Figma design file created by a web designer. The web design was mobile responsive and user friendly which allowed the brand to stand out in a crowded market.
